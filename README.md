# Calculator with History

**Core Java Project | Console-based | Calculation History**

---

## Overview
This is a simple **Calculator Project** developed in **Core Java**.  
It allows users to perform **basic arithmetic operations** and keeps a **history of all calculations** for review.

---

## Technology Stack
- **Java (Core Java)**
- **NetBeans IDE**
- Console-based interface

---

## Features
- Addition, Subtraction, Multiplication, Division  
- Maintain history of previous calculations  
- Prevent division by zero  
- User-friendly console interface  

---

## How to Run
1. Import the project in **NetBeans IDE**  
2. Run `CalculatorCoreJava.java`  
3. Follow the console menu to perform calculations and view history
